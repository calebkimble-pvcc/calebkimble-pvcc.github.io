Chapter 4: Networking and the internet
Term Definition
Network Classification
    LAN: Local Area Network
    MAN: Metropolitan area network
    WAN: Wide Area Network
Network Models
    Bus Network: Where the machines are connected to a common communication line called a bus.
    Star Network: Where one machine is a central focal point and the rest of the machines are connected
Protocalls
    CSMA/CD Ethernet: Carrier Sense, Multiple Access with Collision Detection
    CSMA/CA WiFi: Carrier Sense, Multiple Access with Collision Avoidance
Combining networks
    Repeater: Passess signals back and forth between two bussess
    Bridge: Looks and destination adress and forwards message across connecction only if it is destined for a machine on the other side.
    Switch: Allowes connection to more than one bus.
    Router: special purpose computers used for forwarding messages
    Gateway: The point where the network is linked to the internet
client/Server model
    P2P-Peer to Peer: provides service to annd recieve service from eachother
    Distrubuted system: Software units that execute as processes
    Pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
    Tier 1: very high-speed, high capacity, international WANs
    Tier 2: More reigonal in scope and less potent in their capabilities.
    Tier 3: An access ISP (Internet service provider) is an independent internet.
IP address
    ICANN: Internet Corporation for Assigned Names and Numbers
    DNS: Domain Name System
    TDL(Top-Level-Domain): used to classify the domain. Ex: .edu, .gov
    Country-Code TLDs: Two letter domains for a country. Ex: .ca, .eu
Internet Protocalls
    FTP: File Transfer Protocall
    HTTP: Hyper Text Transfer Protocall
    SMTP: Secure Shell Protocall
World Wide Web
    Uniform Resource Locator
    Client-Side activities: activites performed by the client. Ex: using a browser.
    Server-Side Activites: Activities performed by the server. Ex. hosting a webserver.
Internet Four-Layer Software
    Application layer: software units that use internet communication to execute
    Transport layer: accepts messages (that are made into packets) from the application layer and makes sure they are properly formated for sending
    Network Layer: Decide where the packets go through the internet
    Link Layer: Transfers the packets to its final destination

        Cyber Security
          Malware: Malicious software used to attack the computer
          virus: infects the computer by inserting itself into a program already on the machine
          Worm: an autonomous program that transfers itself through a network
          Trojan horse: a program that enters a computer system disguised as a deirable program
          spyware: collects information about activites at the computer it resides and reports back to the instigator
          phishing: obtaining information by asking for it while usually disguised as some legit agency
          DDoS attack: overloading a computer or server with messages
        Protection
          Firewall: filters traffic passing through
          spam filter: a firewall designed to block unwanted email
          proxy server: software unit that acts as an intermediary between the client and server to protect the client
          antivirus software: detects and removes known viruses and other infections
          cryptography: the study of sending and recieveing messages securely
          public key encryption: involves techniques by which the encryption and decryption algorithms are known
