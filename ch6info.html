High Level programming language Provides more abstractions for the programmer and make it easier for people to read and write a program.
Object Oriented Programming Following this paradigm, a software system is viewed as a collection of units, called objects, each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects.
Scripted Languages A subset of the imperative programming languages is the collection of languages known as scripting languages. These languages are typically used to perform administrative tasks rather than to develop complex programs.
Data Types
  • Integer: Refers to numeric data consisting of whole numbers.
  • Float: Refers to numeric data that might contain values other than whole numbers, probably stored in floating-point notation.
  • String: A data type used in programming, that is used to represent text rather than numbers.
    		•
    Primitive Data Types The data types that are included as primitives in a programming language, such as int for integer and char for character.
    Data Structures The conceptual shape or arrangement of data. For example, text is normally viewed as a long string of characters whereas sales records might be envisioned as a rectangular table of numeric values, where each row represents the sales made by a particular employee and each column represents the sales made on a particular day.
    Python collection data structures
  • List: Lists are used to store multiple items in a single variable. Created using square brackets.
  • Tuple: Used to store multiple items in a single variable, ordered and unchangeable. Written with round brackets.
  • Set: Used to store multiple items in a single variable. It is unordered, unchangeable, and unindexed. Written with curly brackets.
  • Dictionary: Used to store data values in key:value pairs. It is ordered, changeable and do not allow duplicates. Written with curly brackets, and have keys and values.
    		•
    Functions A set of instructions for performing a task that can be used as an abstract tool by other program units.
    Local Variable A variable declared within a function, meaning that it can be referenced only within that function.
    Global Variable Variables whose scopes are not restricted to a particular part of a program.
    Source Progams The program in its original form.
    Object Programs The program in its translated version
