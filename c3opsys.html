Operating systems Terminoligy
Term Definition
Opperating System The software that controls the overall operation of your computer.
Job Queue Storage organization where tasks are put in a first-in first-out fasion.
FIFO First-in, First-out.
Interactive processing Interacting with a program in real time as its computing.
Time-Sharing An operating system that is designed to provide service to multiple users.
Multiprograming Time is divided into intervals and the execution of each job is restricted to one interval.
Application Software Cosnists of the programs for performing tasks specific to the machines use.
System Software Performs tasks that are common to computer systems in genreal.
Untility Software Software that is fundamental to the computer installation but not included in the OS.
Graphical User Interface (GUI) A user interface where you can manipulate objects like files and programs.
Shell An older user interface that communicates to the user through text.
Kernel An internal part of an OS that performs teh basic funtions required by the computer.
Device Driver Software that communicate to controlers to carry out operation from the peripherals.
File Manager Coordinates the use of the computers mass storage and where the files are located.
Memory Manager Coordinates the machines main memory
Virtual memory Shuffling data between the main memory and mass storage.
Boot strapping Transering the OS from mass storage into main memory.
ROM Read-Only Memory
Deadlock When two or more programs are blocked from progressing because they are waiting for a recource allocated to another.
Spooling Holding data for output at a later and more convenient time.
Super user (or Admin) A user who has access to alter settings within the OS.
Multi-factor authentication Needed two or more authentications to access data or an account
